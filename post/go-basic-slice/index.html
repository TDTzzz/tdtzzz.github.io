创建方式

  1. 数组下标(或者切片下标)，比如a:=[5]int{1,2,3,4,5} s:=a[1:3]
  2. 字面量，比如slice:=[]int{1,2,3,4}
  3. make
  4. 直接声明var s []int,注意这里得到的是nil slice
  5. 用new,比如slice:=*new([]int)

// runtime/slice.go
type slice struct {
	array unsafe.Pointer // 指向数组的元素指针
	len   int // 长度 
	cap   int // 容量
}

s[slow:high:max] 从切片s的索引位置low到high所获得的切片(左闭右开),len=high-low,cap=max-low

nil slice和empty slice

  • nil slice: len=0，cap=0，而且指向数组的元素指针指向了nil
  • empty slice: len=0，cap=0，但是指向数组的元素指针指向了底层数组

slice扩容

扩容规则

  • 当slice长度容量小于1024时，每次append超出扩容时，会double容量
  • 当超出1024时，不只是单纯的1.25倍扩充，还有内存对齐的操作，导致会>1.25
  • **注意：**go1.16更改了规则，1.16之前是 if old.len < 1024,1.16后是if old.cap<1024。改成了以cap判断

// go 1.9.5 src/runtime/slice.go:82
func growslice(et *_type, old slice, cap int) slice {
    // ……
    newcap := old.cap
	doublecap := newcap + newcap
	if cap > doublecap {
		newcap = cap
	} else {
		//if old.len < 1024 {  go1.16之前
    if old.cap < 1024 {
			newcap = doublecap
		} else {
			for newcap < cap {
				newcap += newcap / 4
			}
		}
	}
	// ……
	
	capmem = roundupsize(uintptr(newcap) * ptrSize)//内存对齐
	newcap = int(capmem / ptrSize)
}

//内存对齐
// src/runtime/msize.go:13
func roundupsize(size uintptr) uintptr {
	if size < _MaxSmallSize {
		if size <= smallSizeMax-8 {
			return uintptr(class_to_size[size_to_class8[(size+smallSizeDiv-1)/smallSizeDiv]])
		} else {
			//……
		}
	}
    //……
}

const _MaxSmallSize = 32768
const smallSizeMax = 1024
const smallSizeDiv = 8

由上面代码可以看到，固定扩展cap后，还需要进行内存对齐

//扩容问题
func testAppend() {
	s := make([]int, 0)
	oldCap := cap(s)

	for i := 0; i < 2048; i++ {
		s = append(s, i)
		newCap := cap(s)

		if oldCap != newCap {
			fmt.Printf("[%d -> %4d] cap = %-4d  |  after append %-4d cap =%-4d\n", 0, i-1, oldCap, i, newCap)
			oldCap = newCap
		}
	}
}

大于1024后，就没有完全遵循double的扩展规则，1280/1024=1.25,1696/1280=1.35，可以看到1024后的扩容不是1.25，这是因为有 内存对齐

//切片的函数传递问题
func main() {
	s := []int{1, 2, 3}
	fmt.Println(unsafe.Pointer(&s))
	f(s)
	fmt.Println(unsafe.Pointer(&s))
	fmt.Println(s)
}

func f(s []int) {
	fmt.Println(unsafe.Pointer(&s))
	for i := range s {
		s[i]++
	}
}

	fmt.Printf("len=%d,cap=%d", len(s), cap(s))
